# apple-store-availability

## install

```
npm install -g apple-store-availability
```

## usage

```
  Usage: apple-store-availability [options] [zip] [part-numbers...]

  Options:

    -h, --help                    output usage information
    -V, --version                 output the version number
    -w, --watch                   Watch (poll) for changes in availability
    -wi, --watch-interval <n>     The interval (in seconds) to watch (defaults to 120)
    -wq, --watch-quit-on-success  Quit watching once availability is found
    -n, --notify                  Trigger a OS notification on success
```
